nvm  -> picks Node version
Node -> runtime that executes tools/servers
npm  -> installs packages & runs scripts from package.json

React -> UI component library (browser + optional server render)
Next  -> Framework around React (routing, SSR/SSG, API routes, build)

EditorConfig -> for the IDE
SCSS -> css wrapper
TypeScript -> JavaScript wrapper

ESLint -> static analysis for JS
StyleLint -> static analysis for css,scss
